## Overview

This guide provides steps on how to retrieve general information, including user profiles and various table data, using the ClaimMind API.

### Prerequisites
*   An authenticated `access_token`.

<Steps>
<Step title="1. Get User Profile">
Retrieve the profile information for a specific user.

<CodeGroup>
```bash
curl -X GET "{api_base_url}/v1/auth/whoami" \
-H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

```python
import requests

api_base_url = "{api_base_url}"
access_token = "YOUR_ACCESS_TOKEN"

response = requests.get(
    f"{api_base_url}/v1/auth/whoami",
    headers={
        "Authorization": f"Bearer {access_token}"
    }
)
print(f"User Profile: {response.json()}")
```
</CodeGroup>
<Check>
You should receive a `200 OK` response with the user's profile details.
</Check>
</Step>

<Step title="2. Get Specific Claim Data">
Retrieve information for a specific claim using its `claim_id`. This demonstrates how to fetch detailed data for a single entry.

<CodeGroup>
```bash
curl -X GET "{api_base_url}/v1/claims/get/YOUR_CLAIM_ID" \
-H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

```python
import requests

api_base_url = "{api_base_url}"
access_token = "YOUR_ACCESS_TOKEN"

response = requests.get(
    f"{api_base_url}/v1/claims/get/YOUR_CLAIM_ID",
    headers={
        "Authorization": f"Bearer {access_token}"
    }
)
print(f"Claim Details: {response.json()}")
```
</CodeGroup>
<Check>
You should receive a `200 OK` response with the details of the specific claim.
</Check>
</Step>
</Steps>
